Разработать приложение для браузера, для учета работников предприятия:

Сущности:
работник: имя, фамилия, дата рождения, url фото, профессия
профессия: название профессии

В приложении должна быть возможность добавить/изменить/удалить работника, указать его профессию. Возможность добавить/изменить/удалить профессии.

Все приложение должно быть реализовано таким образом, чтобы не происходило обновление страницы (SPA). Нельзя добавить профессии с одинаковыми названиями. Должно быть поле поиска, по фамилии и\или имени.

API для получения юзеров: https://randomuser.me/

Интерфейс в произвольном виде.
